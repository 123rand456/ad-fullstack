## Running locally

1. RECOMMENDED: Create new virtual environment using IDE.
2. In IDE terminal, navigate to `/back/fastApi/price_predictor/` directory
3. Run 
```bash 
pip install --no-cache-dir -r requirements.txt
```
4. Execute command 
```bash 
fastapi run
```
